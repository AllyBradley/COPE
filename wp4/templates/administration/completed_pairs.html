    This report follows a waterfall process where each donor and subsequent kidneys, recipients, and final follow up are assessed through a series of simple tests based on the data recorded.

    Donor
    We start by taking all donor records (as generated by starting of a procurement form) and listing them in order of creation. We count the number of records to produce this first total.
    Organ
    We presume for this trial that each donor produces two kidneys (or otherwise would not have been included), and we count those to get the total (i.e. double the number of donors). To be eligible for the next stage, each kidney must have all three of the following: Preservation set (i.e. has been randomised); Donor is known to have provided for multiple recipients (i.e. both kidneys will not end up in the same person); Organ to be marked as Transplantable (i.e. based on initial inspection upon removal, and recorded as transplantable).
    Allocation
    Eligible kidneys are counted again to provide a total number of allocation, and then required to pass the following two tests to be counted further: Allocated to a project site (i.e. a project site location has been recorded as the transplant hospital in the latest allocation for that kidney; And the latest allocation has been marked as NOT reallocated (i.e. this was the final destination for the organ).
    Recipient
    Once a final allocation has been recorded for an eligible organ, a recipient record is started, and this is then counted as Recipient Total records. We then look for one of the following four dates to be recorded to indicate an operation actually occurred (because we don't ask if the organ was implanted, we have to presume it based on other actions occurring): Knife to Skin, or Anastomosis Started at, or Reperfusion Started at, or Operation Concluded at. The number of recipients with one or more of these dates is counted as the Total operated on. We finally require recipient consent to be recorded, and confirmation that the recipient received only this kidney and no other organs, which further reduces the number of operations that are counted as eligible.
    Eligible
    The counts here are based on both kidneys from one donor passing all the above tests to be a pair, otherwise if one kidney has passed all the tests and not part of a pair, it is counted as a single.
    12 Month Follow Up
    The most complicated category to assess due to weaknesses in the paper protocol, the follow up looks primarily for a recorded Creatinine Clearance to have occurred within the window defined as 365 +/-65 days from the date of randomisation. Where a start date has been recorded on the follow up, we use this to assess if it is in the 130 day window, or late/early. If no start date is recorded, we use today's date to determine if we are in the window, early, or late. Again, we are only assessing eligible kidneys.
